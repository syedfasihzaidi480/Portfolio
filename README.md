## Description

This website serves as Fasih's online resume and portfolio. It provides visitors with information about Fasih's background, skills, projects, and ways to contact him.

## Features

- Responsive design
- Banner section with Fasih's name, designation, and buttons for additional information
- Gallery section displaying Fasih's recent portfolio items
- Slider section showcasing images
- Section displaying Fasih's personal projects
- Section highlighting Fasih's language proficiency skills
- Newsletter subscription form
- Contact information section with Fasih's phone number and email
- Social media links in the footer
